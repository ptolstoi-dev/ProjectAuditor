# ProjectAuditor.Tests

Enthält die automatisierten Unit-Tests (xUnit) für die `ProjectAuditor.Core` Bibliothek. 

Die Tests stellen sicher, dass die Parsing-, Manipulations- und Analyse-Logik (z.B. der `ProjectParser`) für verschiedene XML-Strukturen und Dateiarten korrekt funktioniert und Versionen sicher extrahiert und geschrieben werden können.

## Verwendung

Führe alle Tests über die CLI aus:

```bash
dotnet test
```
